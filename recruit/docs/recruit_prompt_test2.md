
[Context & Base Data]
내 프로젝트 폴더에 사람인에서 크롤링한 회계 직무 채용 공고 데이터(recruit\data\recruit.db)가 있어. 이전 단계에서 탐색적 데이터 분석(EDA)을 통해 도출된 recruit\docs\recruit_prompt_test.md 의 경력, 학력 범주 및 자격증/툴/세부직무 핵심 키워드 35종과 동의어 매핑 데이터를 바탕으로, 전체 데이터를 정제하고 비지도 학습(K-Means Clustering)을 통해 기업 유형을 스스로 발굴한 뒤, 이를 인터랙티브하게 보여주는 Streamlit 대시보드 웹앱(app.py)을 빌드하고 실행해줘.


---

### 요구사항 1: 백엔드 데이터 구조화 및 머신러닝 군집화 파이프라인

1. 동의어 정제 및 변환: '자격요건' 및 '우대사항' 텍스트에서 상기 정의된 동의어 사전 규칙에 따라 단어들을 대표 키워드로 일괄 치환해줘.
2. 바이너리 매트릭스 생성: 공고별로 상기 35종의 핵심 키워드(자격증, 툴, 세부직무)가 포함되어 있으면 1, 없으면 0을 부여하는 독립 변수(Feature) 컬럼들을 생성해줘. 경력과 학력도 범주화 컬럼으로 인코딩해줘.
3. 데이터 기반 기업 유형 분류 (K-Means Clustering):
   - 위에서 생성한 바이너리 매트릭스 피처들을 활용하여 K-Means 군집 분석을 수행해줘.
   - 최적의 군집 개수(K)는 실루엣 스코어를 기준으로 3~4개 사이에서 자동 결정되도록 하거나, 고정값 K=3 또는 4로 설정해줘.
   - 학습 완료 후 각 공고에 군집 번호(`cluster_label`)를 부여하고, 군집별로 어떤 스펙 키워드의 비중(평균값)이 높은지 분석하여 각 군집의 성격(예: Cluster 0 = 인사/총무 멀티태스킹형, Cluster 1 = 세무대리형 등)을 정의하는 프로파일링 딕셔너리를 내부적으로 생성해줘.

### 요구사항 2: Streamlit 대시보드 UI 및 인터랙티브 로직 구현 (app.py)

대시보드 상단에는 "회계 취준생들의 컴활 탈출기: 시장 데이터를 기반으로 한 미스매치 및 허수 지원 필터링 가이드"라는 메인 타이틀을 배치하고 아래 2개의 탭을 구현해줘.

#### 💻 [탭 1: 구직자 탭 - "내 회계 직무 적합도 & 우선순위 리포트"]

1. 유저 입력 UI: 구직자가 자신의 학력, 경력을 라디오/셀렉트 박스로 선택하고, 보유 자격증, 다룰 줄 아는 툴, 경험해 본 세부 직무를 체크박스나 멀티셀렉트로 다중 선택할 수 있게 해줘.
2. 실시간 직무 적합도 점수 산출:
   - 유저가 체크한 항목(1)과 체크하지 않은 항목(0)으로 유저 벡터를 생성해줘.
   - 1,596건의 공고 벡터들과 '코사인 유사도(Cosine Similarity)'를 계산하고, 시장 전체 평점 기준 "내 회계 직무 적합도: OO점"을 상단에 커다란 st.metric 카드로 시각화해줘.
3. 데이터 기반 매칭 기업 유형 안내:
   - 유저 벡터와 가장 높은 유사도를 보인 '머신러닝 군집(Cluster)'이 무엇인지 판별하여, "귀하의 스펙은 데이터 분석 결과 [OOO 유형의 기업]에 가장 잘 맞습니다"라는 가이드를 띄워줘.
4. 부족한 스펙 선점 우선순위 TOP 3 제안:
   - 유저가 매칭된 해당 군집의 기업들이 가장 많이 요구(요구 빈도 비율 상위)하고 있지만, 유저가 현재 체크하지 않은 스펙 키워드 3개를 동적으로 추출하여 "현재 목표 유형에서 가장 필요로 하나 나에게 부족한 스펙 TOP 3"로 화면에 제안해줘.

#### 🏢 [탭 2: 인사팀 탭 - "회계 스펙 미스매치(Gap) 및 JD 가이드"]

1. 유저 입력 UI: 인사담당자가 대시보드에서 데이터가 스스로 분류한 '머신러닝 기업 군집 유형(Cluster)'을 드롭다운으로 선택할 수 있게 해줘.
2. 스펙 갭(Gap) 시각화:
   - 기획 배경을 완벽히 반영하여, 구직자들이 흔히 매달리는 단순 범용 스펙(예: 엑셀/컴활, 전산회계 2급)의 시장 요구 빈도와, 해당 군집의 기업들이 진짜 목말라하는 진성 실무 스펙(예: 급여 계산, 4대 보험, 회계 결산, SAP 등)의 요구 빈도 차이를 좌우 대칭형 바 차트나 Plotly 갭 차트로 시각화해줘.
3. 채용 공고(JD) 가이드라인 제안:
   - 인사팀이 묻지마 허수 지원자를 필터링하고 직무 진정성이 높은 인재를 모으기 위해, 선택한 군집의 핵심 역량을 기반으로 우대사항에 반드시 넣어야 할 '실무 타겟 핵심 행동 키워드 가이드 TOP 3'를 텍스트 박스로 동적 추천해줘. (예: "단순 사무 능력 우대 문구를 삭제하고 '급여 계산 및 4대 보험 실무 가능자'를 명시하세요").

위 백엔드 군집화 모델링과 프론트엔드 Streamlit 웹앱 구현 코딩을 완벽히 결합하여 `app.py`로 저장한 뒤, 안티그래비티 터미널 환경에서 서버를 구동하고 정상 동작 여부까지 최종 검증을 마쳐줘.
