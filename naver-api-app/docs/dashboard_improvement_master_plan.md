# 네이버 API 종합 분석 대시보드 개선 마스터 계획서 (Master Improvement Plan)

본 마스터 계획서는 기획자(Planner)와 UX/UI 리서처(UX/UI Researcher) 두 서브에이전트의 심층 분석과 협업 결과를 바탕으로 작성되었습니다. 기존 `naver-api-app` 대시보드를 단순한 '데이터 조회용 유틸리티'에서 비즈니스 가치가 높은 **'시장 분석 및 마케팅 전략 수립 플랫폼'**으로 진화시키기 위한 기능적, 시각적, 기술적 개선 계획을 포괄적으로 수립합니다.

---

## 1. 종합 개선 방향 요약

본 개선 작업의 최우선 과제는 다음과 같습니다.
1. **의사결정에 활용 가능한 인사이트 도출**: 단순 데이터 나열을 넘어 형태소 분석 및 긍/부정 감성 분석을 통해 사용자에게 해석된 가치를 제공합니다.
2. **비즈니스 데이터 확장**: 네이버 오픈 API의 한계(상대 지수)를 넘어서기 위해 네이버 검색광고 API를 연동하여 실제 검색량과 경쟁 강도를 파악합니다.
3. **일관성 있고 직관적인 UX/UI**: 대시보드의 테마 컬러를 통일하고, 정보의 위계를 재정리하며, 잦은 화면 리렌더링과 API 호출 비효율성을 방지하는 기술적 장치를 도입합니다.
4. **최종 보고서 생산성 향상**: 분석 결과를 원클릭으로 격식 있는 Excel(`.xlsx`) 및 PDF 보고서로 내려받아 내부/대외 공유에 즉시 활용할 수 있도록 지원합니다.

---

## 2. 세부 개선 계획

### 2.1. 기능 및 비즈니스 고도화 (Planner's Proposal)
*   **[데이터 가공] 한국어 자연어 처리(NLP) 고도화**:
    *   Pure-Python 기반의 가볍고 강력한 한국어 형태소 분석기인 `kiwipiepy`를 탑재하여 블로그, 카페글, 뉴스 요약 텍스트에서 불필요한 은는이가 등의 조사를 거르고 의미 있는 형태소(명사, 형용사, 동사)를 추출합니다.
    *   추출된 텍스트의 맥락에 기반한 긍/부정 감성 점수를 분류하고 감성 비율 및 주요 원인 단어 비교 바 차트를 제공합니다.
*   **[다차원 분석] 성별 x 연령대 교차 타겟팅 분석**:
    *   기존의 단편적 성별/연령대 필터를 결합하여 성별과 연령대가 교차하는 2차원 히트맵(Heatmap)을 시각화합니다.
    *   핵심 타겟층(예: "30대 여성층이 전체 클릭의 45% 차지")을 텍스트 카드로 자동 파싱하여 요약하는 페르소나 자동화 기능을 구현합니다.
*   **[기능 확장] 네이버 검색광고 API (`keywordstool`) 연동**:
    *   인증 정보를 입력받아 네이버 광고 시스템에서 연관 키워드를 제안받고, 실제 월간 검색수(PC/모바일), 월간 클릭수, 경쟁 강도(높음/중간/낮음) 정보를 수집합니다.
    *   검색량이 우수하지만 경쟁 강도가 낮은 블루오션 키워드를 선별하는 필터 뷰를 구현합니다.
*   **[산출물 출력] 포맷팅된 Excel 및 PDF 보고서 내보내기**:
    *   셀 스타일링과 탭 분류가 들어간 격식 있는 Excel 파일 생성.
    *   화면상의 시각화 차트 이미지와 핵심 텍스트 요약을 A4 포맷에 얹은 종합 시장 분석 보고서 PDF 실시간 생성 기능 제공.

### 2.2. UX/UI 및 기술 개선 (UX/UI Researcher's Report)
*   **[정보 설계] 글로벌/로컬 제어 구조 분리 및 분석 실행 버튼 도입**:
    *   사이드바에는 입력 빈도가 낮은 '네이버 API 설정'만 보존하고 주요 검색 키워드 및 기간은 메인 화면 상단으로 분리합니다.
    *   사용자가 타이핑하거나 설정을 바꿀 때마다 자동으로 페이지가 새로고침되는 버벅임을 제어하기 위해 `st.button("🔍 데이터 분석 실행")`을 도입하여 입력 완료 후 명시적 호출을 지원합니다.
*   **[반응형 레이아웃] 컬럼 가로폭 최적화 및 이상치 하이라이팅**:
    *   가로 그리드 분할 시 3열을 초과하지 않도록 구성하여 소형 해상도에서의 시각 요소 찌그러짐 현상을 방지합니다.
    *   `shopping_search.py`에서 불필요한 차트 중복(Box Plot과 Histogram)을 Violin Plot 또는 Box Plot 단일 차트로 일원화합니다.
    *   IQR 사분위수 분석 결과로 검출된 가격 이상치(Outliers) 상품들에 대해 데이터프레임 내에서 경고 이모지(⚠️)나 배경색으로 시각적 피드백을 강화합니다.
*   **[비주얼 시스템] 브랜드 테마 단일화**:
    *   제각각이었던 Plotly의 컬러 스케일(Viridis, Cividis 등)을 'Naver Forest Green (#03C75A)'과 'Deep Teal (#0E7490)'을 주조색으로 하는 일관성 있는 비즈니스 분석 톤앤매너로 통일합니다.
*   **[성능 최적화] 지연 로딩(Lazy Loading) 및 캐싱 세분화**:
    *   `integrated_analysis.py` 등 멀티 탭 구조 페이지에서 사용자가 실제로 탭을 활성화하는 시점에만 해당 탭의 API를 호출하도록 비동기 로직을 리팩토링합니다.
    *   API 데이터의 갱신 빈도에 맞춰 트렌드 정보(24시간, `ttl=86400`)와 실시간 검색글 정보(30분, `ttl=1800`)의 캐시 만료 기준(TTL)을 차등 적용합니다.
